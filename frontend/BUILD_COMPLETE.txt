시작하기
================================================================================

1. 의존성 설치
   $ cd frontend
   $ npm install

2. 개발 서버 실행
   $ npm run dev
   → http://localhost:5173

3. 프로덕션 빌드
   $ npm run build
   → dist/ 디렉토리

4. 빌드 결과 미리보기
   $ npm run preview
